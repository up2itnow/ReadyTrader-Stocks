## MCP client configs (copy/paste)

These files are convenience snippets so a new user can wire ReadyTrader into an MCP-capable agent quickly.

- **Agent Zero**: `configs/agent_zero.mcp.yaml`
- **Claude Desktop / generic clients**: `configs/claude_desktop.mcp-server-config.json`

All configs default to `PAPER_MODE=true` and use the server key `readytrader_crypto`.
